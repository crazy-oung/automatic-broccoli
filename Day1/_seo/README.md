# SEO
## SEO (Search Engine Optimization)?
- 검색엔진최적화
- 웹 페이지 검색엔진이 자료를 수집하고 순위를 매기는 방식에 맞게 웹 페이지를 구성해서 검색 결과의 상위에 나올 수 있도록 하는 작업
- SEO가 기본적으로 구글봇같은 크롤링해주는 봇들이 데이터를 수집하게 되는데 
  - [meta tag](#meta-tag)가 상당히 중용한 요소를 차지
