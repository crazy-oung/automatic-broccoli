# 로드밸런싱
## 로드 밸런싱 (load balancing, 부하분산)
- 다수의 서버(중앙처리장치 혹은 저장장치와 같은 컴퓨터)에게 작업을 나누는 것
  - 가용성 및 응답시간을 최적화 시킬 수 있음
- 다수의 서버를 가지고 한 가지 종류의 인터넷 서비스를 지원
- 트래픽이 많은 웹 사이트, IRC 네트워크, FTP 사이트, NNTP 서버 그리고 DNS서버에 적용
